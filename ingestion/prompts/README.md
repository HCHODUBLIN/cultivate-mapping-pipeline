# Prompts - Design Patterns and Governance

This directory contains prompt design patterns only.
Full prompt text is intentionally excluded from this public repository.

## Included in this repository

- Template structure and variable schema
- Versioning rules and change log style
- Public-safe prompt governance notes

## Excluded from this repository

- Full production prompt text
- Prompt internals that expose private operational logic

## Source of full prompt text

See paper appendices for full versions:
- Appendix E (v2.0.0)
- Appendix F (v3.0.0)

## Principle

Keep a strict boundary over time: repository stores prompt patterns,
while full text remains in the linked publication appendices.
